# 📊 Calculadora de IMC

Projeto simples em Python que calcula o Índice de Massa Corporal (IMC) e exibe a classificação de acordo com a OMS.

---

## 🚀 Demonstração

```bash
Informe o peso em kg: 70
Informe a altura em metros: 1.75

Seu IMC é: 22.86
Você está no peso ideal
```

---

## 🛠️ Funcionalidades
- Cálculo automático do IMC
- Classificação por faixa (abaixo do peso até obesidade grau 3)
- Validação de entrada de dados
- Tratamento de erros (valores inválidos)
- Loop para múltiplos cálculos

---

## ▶️ Como executar
Clone o repositório:
```bash
git clone https://github.com/OMathgr/Calculadora_IMC.git
```
Acesse a pasta:
```bash
cd Calculadora_IMC
```
Execute o programa:
```bash
python IMC.py
```
